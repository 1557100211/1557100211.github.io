SVN迁移GIT

2020-05-18

迁移过程

  1. 拉取svn代码并转为git（输入用户名密码）
    git svn clone   http://xxxx.com/project
  2. 关联远程仓库(保证仓库是新建的并且没有任何提交)
    cd project/
git remote add origin http://xxxx.com/project
  3. 提交
    git push -u origin master
